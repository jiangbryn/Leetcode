### 题目
Given a binary tree, return the inorder traversal of its nodes' values.
**Follow up:** Recursive solution is trivial, could you do it iteratively?
### 我的想法
**递归法**
```java
class Solution {
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> arr = new ArrayList<>();
        if(root == null) return arr;
        inorderTraversal(root, arr);
        return arr;
    }
    private void inorderTraversal(TreeNode root, List arr){
        if(root.left != null) inorderTraversal(root.left, arr);
        arr.add(root.val);
        if(root.right != null) inorderTraversal(root.right, arr);
    }
}
```

**迭代(Stack)**
总是先找到最左边的，再把根节点pop出来，如果有右节点，把当前节点设为右节点。
写出来总是不对。。。
```java
class Solution {
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> arr = new ArrayList<>();
        if(root == null) return arr;
        Stack<TreeNode> stack = new Stack<>();
        stack.push(root);
        
        while(stack != null || root != null){  
            while(root.left != null){  //错误：pop出根节点之后又回到了最左端
                stack.push(root);
                root = root.left;
            }
            root = stack.pop();
            arr.add(root.val);
            if(root.right != null) root = root.right;
        }
        return arr;
    }
}
```

**Morris**
```java
class Solution {
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> arr = new ArrayList<>();
        if(root == null) return arr;
        Stack<TreeNode> stack = new Stack<>();
        TreeNode prev;
        
        while (root != null)
        {
            if(root.left == null) {   
                arr.add(root.val);      //访问左结点
                root = root.right;
            }
            else{
                // find predecessor
                prev = root.left;
                while (prev.right != null && prev.right != root)
                    prev = prev.right;  //前驱结点是左子树中最右边的叶子结点

                if(prev.right == null){ //暂未成环
                    prev.right = root;  //成环
                    root = root.left;   //继续访问左子树
                }
                else{                   //已成环，且已完成退回
                    prev.right = null;  //解除环
                    arr.add(root.val);  //访问中间点
                    root = root.right;  //访问右子树
                }
            }
        }
        return arr;
    }
}
```
### 解答
leetcode solution: Using Stack
```java
class Solution {
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> arr = new ArrayList<>();
        if(root == null) return arr;
        Stack<TreeNode> stack = new Stack<>();
        while(root!=null || !stack.empty()){
            while(root!=null){
                stack.add(root);
                root = root.left;
            }
            root = stack.pop();
            arr.add(root.val);
            root = root.right;
        }
        return arr;
    }
}
```
